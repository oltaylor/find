# Find - Velocity Plugin
One simple command that can be very helpful across a Velocity network.
---
## Usage
```/find <username>``` - outputs the server on which a player can be found (if they can be found, that is!). If found, you can click on the message to teleport to the relevant server.
## Permissions
```find.find``` permission is required to use the command.
## Compatibility
Tested with Velocity 4.1.0
## Installation
**THIS IS NOT A PAPER PLUGIN. THIS IS TO BE INSTALLED ONTO A VELOCITY PROXY ONLY**<br>
Simply drag the [latest ```.jar```](https://github.com/oltaylor/find/releases) file into the Velocity plugins folder.
